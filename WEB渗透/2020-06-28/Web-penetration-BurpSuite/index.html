BurpSuite 简单介绍

  • Burp Suite 是用于攻击 web 应用程序的集成平台，包含了许多工具。

  • Burp Suite 为这些工具设计了许多接口，以加快攻击应用程序的过程。

  • 所有工具都共享一个请求，并能处理对应的 HTTP 消息、持久性、认证、代理、日志、警报。
